#### 3.4. 最終結果画面

* **目的:** 全ての出題が終わり、最終的な優勝チームを発表する。
* **対象ロール:** 全員
* **表示要素:**
    * ヘッダー: 「ゲーム終了！」
    * タイトル: 「最終結果」
    * 最終スコアボード (全チームの最終ポイントと順位)
    * 優勝チーム表示:
        * 「優勝は… **[優勝チーム名]**！！！」 (大きく、派手なエフェクト)
    * ボタン: 「新しいゲームを開始」(司会者のみ)
* **UIイメージ:**
  ```
  +------------------------------+
  |          ゲーム終了！        |
  +------------------------------+
  |          最終結果            |
  +------------------------------+
  | 1位: チームA    150pt        |
  | 2位: チームC    120pt        |
  | 3位: チームB    100pt        |
  | ...                          |
  +------------------------------+
  |     優勝は…                  |
  |       チームA !!!            |
  |     (紙吹雪エフェクト)       |
  +------------------------------+
  | (司会者のみ)                 |
  |   [ 新しいゲームを開始 ]     |
  +------------------------------+
  ```
