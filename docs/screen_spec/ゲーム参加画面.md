#### 3.1. ゲーム参加画面

* **目的:** ユーザーがゲームセッションに参加するための入り口。
* **対象ロール:** 全員 (司会者は管理ログインもここから)
* **表示要素:**
    * アプリタイトル: 「ウソホントゲーム」
    * 入力フィールド:
        * 「ゲームコード / セッションID」 (6桁程度の英数字)
        * 「ニックネーム」 (必須)
    * ボタン: 「ゲームに参加」
    * リンク: 「司会者ログインはこちら」 (司会者専用)
* **UIイメージ:**
  ```
  +------------------------------+
  |         ウソホントゲーム     |
  |                              |
  |   [ ゲームコード ]           |
  |   [ ニックネーム ]           |
  |                              |
  |   [ ゲームに参加 ]           |
  |                              |
  |   司会者ログインはこちら     |
  +------------------------------+
  ```
