#### 3.2. 司会者 - ゲーム管理画面 (初期設定時のみ表示)

* **目的:** 司会者がゲームの初期設定（チーム作成、参加者割り当て、エピソード確認など）を行う。
* **対象ロール:** 司会者
* **表示要素:**
    * ヘッダー: 「ゲーム管理」
    * セッション情報: 「ゲームコード: [XXXXXX]」
    * セクション1: 参加者一覧とチーム割り当て
        * 参加者リスト (ニックネームと所属チームを表示。未割り当ては「未所属」)
        * チームリスト (A, B, C...)。参加者をチームにドラッグ＆ドロップ（または選択式）で割り当て。
        * ボタン: 「チーム確定」
    * セクション2: エピソード確認 (任意)
        * 各参加者が登録したエピソード数を確認。
    * セクション3: ゲーム設定
        * 出題順序の設定 (チームA→B→C...のリスト表示、並べ替え可能)
        * ポイントルール確認 (変更は高頻度でないため、確認のみ)
    * ボタン: 「ゲーム開始」 (全ての準備が整ったらアクティブ化)
* **UIイメージ:** (スクロール前提)
  ```
  +------------------------------+
  |         ゲーム管理           |
  +------------------------------+
  | ゲームコード: XXXXXX         |
  +------------------------------+
  | ■ チーム設定                |
  |   未所属: プレイヤー1, プレイヤー5|
  |   チームA: プレイヤー2, プレイヤー4|
  |   チームB: プレイヤー3       |
  |   [ チーム確定 ]             |
  +------------------------------+
  | ■ 出題順序                  |
  |   1. チームA                 |
  |   2. チームB                 |
  |   3. チームC                 |
  |   ... (並べ替えアイコン)     |
  +------------------------------+
  |                              |
  |   [ ゲーム開始 ]             |
  +------------------------------+
  ```
