#### 3.3. メインゲーム画面 (共通ダッシュボード)

* **目的:** ゲームの進行状況、スコアボードを常に表示し、各フェーズに応じた操作を提供する。
* **対象ロール:** 全員 (司会者と参加者で表示・操作内容が変化)
* **表示要素:**
    * ヘッダー: アプリタイトル、現在のターン (例: 「Aチームの番」)
    * スコアボード:
        * 各チーム名と現在のポイントをリアルタイム表示。
        * 現在の出題チームはハイライト表示。
    * ゲーム進行表示エリア:
        * 現在のフェーズに応じたコンテンツが動的に表示される。

##### 3.3.1. メインゲーム画面 - 出題準備フェーズ (参加者: エピソード登録)

* **目的:** 参加者が自分の3つのエピソードを登録し、「ウソ」を設定する。
* **対象ロール:** 参加者 (特に代表者となる可能性のある全員)
* **表示要素:**
    * ヘッダー: 「エピソード登録」
    * 指示テキスト: 「自分に関する3つのエピソードを登録してください。1つはウソです。」
    * 入力フォーム:
        * エピソード1: [テキスト入力欄]
        * エピソード2: [テキスト入力欄]
        * エピソード3: [テキスト入力欄]
    * ラジオボタン/チェックボックス: 各エピソードの下に「これがウソ！」を選択するUI。
    * ボタン: 「登録/更新」
    * 注意書き: 「ウソの情報は他の人には見えません」
* **UIイメージ:**
  ```
  +------------------------------+
  |         エピソード登録       |
  +------------------------------+
  | 3つのエピソードを登録       |
  | 1つはウソです！             |
  +------------------------------+
  | エピソード1:                 |
  | [ 長文テキストエリア ]       |
  | ( ) これがウソ！             |
  +------------------------------+
  | エピソード2:                 |
  | [ 長文テキストエリア ]       |
  | ( ) これがウソ！             |
  +------------------------------+
  | エピソード3:                 |
  | [ 長文テキストエリア ]       |
  | ( ) これがウソ！             |
  +------------------------------+
  |   [ 登録/更新 ]              |
  +------------------------------+
  ```

##### 3.3.2. メインゲーム画面 - 出題フェーズ

* **目的:** 出題者がエピソードを発表し、他のチームが内容を確認する。
* **対象ロール:** 全員
* **表示要素:**
    * ヘッダー: 「[現在の出題チーム名]のターン」
    * 出題者名: 「出題者: [代表者ニックネーム]」
    * エピソードリスト:
        * 1.  [エピソード1]
        * 2.  [エピソード2]
        * 3.  [エピソード3]
    * (司会者のみ) ボタン: 「回答タイム開始」
* **UIイメージ:**
  ```
  +------------------------------+
  |         Aチームのターン      |
  +------------------------------+
  | 出題者: プレイヤーA1         |
  +------------------------------+
  | 1. 私は世界一周したことがある|

  +------------------------------+
  | (司会者のみ)                 |
  |   [ 回答タイム開始 ]         |
  +------------------------------+
  ```

##### 3.3.3. メインゲーム画面 - 回答フェーズ

* **目的:** 回答チームが相談し、「ウソ」を投票する。
* **対象ロール:** 回答者 (司会者は進行管理、出題者は待機)
* **表示要素:**
    * ヘッダー: 「回答タイム - [現在の出題チーム名]」
    * タイマー: 「残り時間: XX秒」 (カウントダウン表示)
    * 出題エピソードリスト:
        * 1.  [エピソード1]
        * 2.  [エピソード2]
        * 3.  [エピソード3]
    * 投票UI (回答者のみ):
        * ラジオボタン: 「1」「2」「3」
        * ボタン: 「このエピソードに投票」 (投票済みの場合は「投票済み」表示)
    * (司会者のみ) ボタン: 「回答締め切り」「正解発表へ」
    * (司会者のみ) 回答状況: 「チームB: 投票済み」「チームC: 未投票」
* **UIイメージ:**
  ```
  +------------------------------+
  |       回答タイム - Aチーム   |
  +------------------------------+
  | 残り時間: 01:23              |
  +------------------------------+
  | 1. 私は世界一周したことがある|
  | 2. 私はラーメン二郎を全店制覇した|
  | 3. 私はUFOを見たことがある   |
  +------------------------------+
  | (あなたのチームの回答)       |
  | O 1  O 2  O 3                |
  |   [ このエピソードに投票 ]   |
  +------------------------------+
  | (司会者のみ)                 |
  |   チームB: 投票済み          |
  |   チームC: 未投票            |
  |   [ 回答締め切り ]           |
  |   [ 正解発表へ ]             |
  +------------------------------+
  ```

##### 3.3.4. メインゲーム画面 - 結果発表フェーズ

* **目的:** 正解を発表し、ポイントを計算・表示する。
* **対象ロール:** 全員
* **表示要素:**
    * ヘッダー: 「結果発表 - [現在の出題チーム名]」
    * 出題者名: 「出題者: [代表者ニックネーム]」
    * 「ウソ」のエピソード表示:
        * 「ウソはこれでした！」
        * **[〇. [ウソのエピソード]]** (大きく、色を変えて表示)
    * 各チームの回答と得点表示:
        * チームB: 選択 [X] -\> **+10pt** (正解) / **+0pt** (不正解)
        * チームC: 選択 [Y] -\> **+0pt** (不正解) / **+10pt** (正解)
    * 出題チームの獲得ポイント:
        * 「[出題チーム名]は、Xチームを騙して **+XXpt** 獲得！」
    * 更新されたスコアボード (リアルタイム更新)
    * (司会者のみ) ボタン: 「次の出題へ」「ゲーム終了」
* **UIイメージ:**
  ```
  +------------------------------+
  |       結果発表 - Aチーム     |
  +------------------------------+
  | 出題者: プレイヤーA1         |
  +------------------------------+
  | ウソはこれでした！           |
  | 2. 私はラーメン二郎を全店制覇した|
  +------------------------------+
  | チームB: 選択 2 -> +10pt (正解!)|
  | チームC: 選択 1 -> +0pt (残念)|
  +------------------------------+
  | Aチームは1チームを騙して     |
  |        +5pt 獲得！           |
  +------------------------------+
  | (更新されたスコアボード)     |
  +------------------------------+
  | (司会者のみ)                 |
  |   [ 次の出題へ ]             |
  |   [ ゲーム終了 ]             |
  +------------------------------+
  ```
